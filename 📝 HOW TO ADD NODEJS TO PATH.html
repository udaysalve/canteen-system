After installing Node.js, Windows needs to know WHERE to find it. The PATH variable tells Windows where to look for programs.

Without this: Commands like node and npm won't work.

üöÄ METHOD 1: Automatic (EASIEST!)

I've created a script that does this automatically for you!

1 Run the script:
Double-click: ADD_NODEJS_TO_PATH.bat

This will automatically find Node.js and add it to your PATH.

2 Close all Command Prompt windows

Any open terminals won't see the new PATH yet.

3 Test it:

Open a NEW Command Prompt and type:

You should see: v24.11.0

4 Run your system:
Double-click: FIX_AND_START.bat

Your servers will now start successfully!

üîß METHOD 2: Manual (If Automatic Fails)

1 Find Node.js installation path

Node.js is usually installed at one of these locations:

C:\Program Files\nodejs
C:\Program Files (x86)\nodejs
%LOCALAPPDATA%\Programs\nodejs

Check which one exists on your computer.

2 Open System Properties

Press Windows Key + R, then type:

sysdm.cpl

Press Enter

3 Go to Environment Variables
  • Click the "Advanced" tab
  • Click "Environment Variables" button at the bottom
4 Edit the Path variable
  • In the "User variables" section (top half)
  • Find and select "Path"
  • Click "Edit"
5 Add Node.js path
  • Click "New"
  • Type: C:\Program Files\nodejs
  • (Or whatever path you found in Step 1)
  • Click "OK"
6 Save and close
  • Click "OK" on Environment Variables window
  • Click "OK" on System Properties window
7 Restart Command Prompt

Close all Command Prompt windows and open a new one.

8 Test it:
node --version

Should show: v24.11.0

‚úÖ How to Verify It's Working

After adding Node.js to PATH, test these commands in a NEW Command Prompt:

Test Node.js:

Expected output: v24.11.0

Test NPM:
npm --version

Expected output: 11.6.1 (or similar)

Test where Node.js is:
where node

Should show the path to node.exe

üîç Common Issues

Issue: "node is not recognized"
Solution:
  • Make sure you opened a NEW Command Prompt after adding to PATH
  • Or restart your computer
Issue: Can't find Node.js installation folder
Solution:
  • Search for "node.exe" in Windows Explorer
  • Use the folder containing node.exe
Issue: Permission denied when editing PATH
Solution:
  • Right-click ADD_NODEJS_TO_PATH.bat
  • Select "Run as Administrator"
